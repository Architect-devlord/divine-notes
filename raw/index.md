# Raw Notes Index

Devlord's own original notes — unedited, unpolished, exactly as typed before any LLM synthesis pass touched them. This is temp/working space Devlord manages directly. Treat it as fluid and personally owned, not a stable part of the wiki's architecture.

**Boundary rule**: files in this folder may link to each other freely. Nothing here links out to `wiki/`, and nothing in `wiki/` links directly to a specific file in here — anything outside `raw/` that wants to point here points at *this file*, never at an individual note below. That's what lets Devlord rename, split, merge, or delete anything in `raw/` without breaking a single link anywhere else in the vault. See `CLAUDE.md` at the vault root for the full rule.

(Links below use the full `raw/` path explicitly — several of these share a filename with an unrelated page over in `wiki/design/`, e.g. `raw/Creaking.md` vs. `wiki/design/Creaking.md`. A bare `[[Creaking]]` would be ambiguous; `[[raw/Creaking|Creaking]]` isn't.)
