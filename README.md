# Divine Notes

Comprehensive documentation and obsidian notes for the Divine World project, serving as the central knowledge repository for architecture, design decisions, development timeline, and project features. You can clone this repo and use the knowledge base for development and maintainance using any AI coding agent.

## Contents

- **DivineWorld.md** - Overview and core concepts of the Divine World project
- **CLAUDE.md** - AI-assisted development notes and interactions
- **Code Timeline.md** - Historical development timeline and code evolution
- **Journey.md** - Project journey and milestones
- **LLM_Wiki_Pattern.md** - Pattern documentation for LLM integration
- **index.md** - Navigation index for documentation
- **log.md** - Development log and updates

## Folders

- **wiki/** - Structured wiki documentation
- **raw/** - Raw notes and reference materials

## Purpose

This repository documents the architecture, design decisions, and development process of the Divine World project, making it easier for developers to understand the project structure, technical decisions, and feature implementations.

## Getting Started

Start with `index.md` for an overview of all documentation, or visit `DivineWorld.md` for the project overview.
