DivineWorld is an experiment to create AI agents that can mimic human evolution and adaptability and can work on multiple environments() in multiple forms without expensive retraining. Its agents have the capabilities such as online tokenizer which helps the agents to learn and adapt on the fly without retraining from scratch.

# A quick intro to what happens in here :

It starts something like this you enter the modded server and you will be greeted by an oracle that will briefly introduce you to the systems of the world and then when you interact with it you will recieve 3 books. Among them is "Genesis" a book that when pressed to the ground spawns 2 AI agents named Adam (male) and Eve (female) and which can be taught speech in game using the oracle's teach function. This oracle is powered by an ollama agent and the natural curiosity of the agents can make the agents learn about the world.

If the world lore feels bland and you need something to spice up things you can summon gods these are essentially minecraft bosses which are controlled by AI agents and which have additional abilities. The recommended way to add them to the world is by using commands which you'll find about in one of the books that the oracle gives you and then disconnect the gods from the server and then run them from their executables and then use chat mode to give them access to the internet and certain websites so that it can learn from it basically they get a head start in the knowledge base and while that you can also chat with it to teach them about their abilities and etc as required (This can be done for the normal agents as well but not preferred because of lore) then make them join the server again they can interact with the other normal agents and shape up the culture of the world.

To make things more interesting the agents are also given the ability to reproduce. Male agents and female agents can breed with each other and with the God agents as well. Moreover the God agents can reproduce with the other God agents as well. Currently breeding only produces normal agents but demi-god agents and new age god agents are planned as well. 

If the world feels out of control or unsatisfactory you can trigger Divine Reset that kills and erases all of the agents except the ones that are spared and then again 2 new agents Adam and Eve spawn again to start the cycle of life again.
